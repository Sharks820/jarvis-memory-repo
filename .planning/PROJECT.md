# PROJECT

## Name
Jarvis Local-First Engine

## Objective
Build a secure, high-intelligence personal agent stack that runs primarily on local hardware, learns through memory and controlled adaptation, and supports optional cloud escalation for hard tasks.

## Constraints
- Local-first by default.
- No unsafe self-modifying model behavior.
- Strong auditability for all high-impact actions.
- Keep operational cost low and predictable.

## Deployment Decision (2026-02-22)
- Launch primary runtime on desktop PC.
- Use weaker laptop later as secondary/support node.
