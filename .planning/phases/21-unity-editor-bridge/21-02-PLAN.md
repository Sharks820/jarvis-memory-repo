Add the UnityTool class to `engine/src/jarvis_engine/agent/tools/unity_tool.py`:

**BridgeState enum:** DISCONNECTED, CONNECTED, WAITING_FOR_BRIDGE

**UnityTool class:**
- `__init__(self, port: int = 8091)` — set state=DISCONNECTED, create asyncio.Event for ready signal, store port, init _ws=None, _request_id counter, _listener_task=None
- `async connect(self)` — use `websockets.asyncio.client.connect(f"ws://localhost:{self._port}/jarvis")`, set state=CONNECTED, set ready_event, start _listen_for_heartbeat as background task
- `async disconnect(self)` — cancel listener task, close WS, set state=DISCONNECTED
- `async call(self, method: str, params: dict | None = None) -> Any`:
  - If DISCONNECTED: raise ConnectionError
  - If WAITING_FOR_BRIDGE: await asyncio.wait_for(ready_event.wait(), timeout=30.0)
  - Increment _request_id
  - Build JSON-RPC request: {"jsonrpc": "2.0", "id": str(_request_id), "method": method, "params": params or {}}
  - Send via _ws.send(json.dumps(request))
  - Receive raw = await _ws.recv()
  - Parse response; if "error" in response: raise RuntimeError(response["error"]["message"])
  - Return response.get("result")
- `async write_script(self, rel_path: str, content: str) -> Any`:
  - Call _assert_in_jail(rel_path) — PRIMARY security gate
  - Call _assert_safe_code(content) — PRIMARY security gate
  - Set state=WAITING_FOR_BRIDGE, clear ready_event
  - Return await self.call("WriteScript", {"path": rel_path, "content": content})
- `async compile(self) -> Any`:
  - Return await self.call("CompileProject", {})
- `async create_project(self, project_path: str) -> Any`:
  - Return await self.call("CreateProject", {"path": project_path})
- `async _listen_for_heartbeat(self)`:
  - Background task: async for raw in self._ws, parse JSON
  - If msg.get("status") == "ready": set state=CONNECTED, set ready_event
  - On ConnectionClosed: set state=DISCONNECTED
- `@staticmethod get_tool_spec() -> ToolSpec`:
  - Return ToolSpec with name="unity", description="Execute Unity Editor commands via WebSocket bridge", parameters schema covering method+params, execute=self.call (or a wrapper)
