Create three new modules in the knowledge package:

**knowledge/locks.py -- FactLockManager:**
- Constants: `LOCK_THRESHOLD_CONFIDENCE = 0.9`, `LOCK_THRESHOLD_SOURCES = 3`.
- Constructor takes the SQLite connection (`db`) and the write lock from MemoryEngine.
- `should_auto_lock(node: dict) -> bool`: Check if confidence >= 0.9 AND len(json.loads(sources)) >= 3. Returns True if both conditions met.
- `lock_fact(db, node_id, locked_by='auto') -> bool`: Atomic UPDATE kg_nodes SET locked=1, locked_at=datetime('now'), locked_by=? WHERE node_id=? AND locked=0. Use write_lock. Return True if rowcount > 0.
- `owner_confirm_lock(db, node_id) -> bool`: Calls lock_fact with locked_by='owner'. This bypasses threshold checks -- owner can lock any fact.
- `unlock_fact(db, node_id) -> bool`: UPDATE kg_nodes SET locked=0, locked_at=NULL, locked_by=NULL WHERE node_id=? AND locked=1. Return True if rowcount > 0.
- `check_and_auto_lock(db, node_id) -> bool`: Load node, call should_auto_lock, if True call lock_fact. This is called after every add_fact that updates confidence or sources.

**knowledge/contradictions.py -- ContradictionManager:**
- Constructor takes db connection and write_lock.
- `list_pending(limit=20) -> list[dict]`: SELECT from kg_contradictions WHERE status='pending' ORDER BY created_at DESC LIMIT ?. Returns list of dicts with all fields.
- `list_all(status=None, limit=50) -> list[dict]`: SELECT with optional status filter.
- `resolve(contradiction_id, resolution, merge_value='') -> dict`: Resolution must be one of 'accept_new', 'keep_old', 'merge'.
  - 'accept_new': UPDATE kg_nodes SET label=incoming_value, locked=0 WHERE node_id=contradiction.node_id. The new value replaces the old and becomes unlocked (needs re-confirmation to lock again). UPDATE kg_contradictions SET status='resolved', resolution='accept_new', resolved_at=datetime('now').
  - 'keep_old': Just UPDATE kg_contradictions SET status='resolved', resolution='keep_old', resolved_at=datetime('now'). No node change.
  - 'merge': UPDATE kg_nodes SET label=merge_value WHERE node_id=contradiction.node_id. UPDATE kg_contradictions SET status='resolved', resolution='merge', resolved_at=datetime('now').
  - All resolutions append to the node's history JSON array: {"action": resolution, "previous_value": existing_value, "new_value": ..., "resolved_at": ...}.
  - Use write_lock for all writes.
  - Return dict with success, node_id, resolution, message.

**knowledge/regression.py -- RegressionChecker:**
- Constructor takes a KnowledgeGraph instance.
- `capture_metrics() -> dict`: Build NetworkX DiGraph via `kg.to_networkx()`. Return dict with: node_count, edge_count, locked_count (nodes where locked=True), graph_hash (via `nx.weisfeiler_lehman_graph_hash(G, node_attr='label', edge_attr='relation', iterations=3, digest_size=16)`), captured_at (ISO timestamp). Handle empty graph case: graph_hash should be a consistent empty hash.
- `compare(previous: dict, current: dict) -> dict`: Compare two metric snapshots. Report discrepancies:
  - node_loss: current.node_count < previous.node_count
  - edge_loss: current.edge_count < previous.edge_count
  - locked_fact_loss: current.locked_count < previous.locked_count (severity: critical)
  - graph_hash_change: hash changed without count increase (severity: warn, could indicate modification rather than loss)
  - Return dict with status ('pass', 'warn', 'fail'), discrepancies list, previous/current metrics.
- Handle "no previous snapshot" case explicitly: return status='baseline', message='Baseline established, no comparison available'.

**handlers/knowledge_handlers.py -- 5 handler classes:**
Each handler follows the same pattern: constructor takes `root: Path` and `kg: KnowledgeGraph | None = None`. If `kg` is None, return an error result gracefully (knowledge graph not available).

- `KnowledgeStatusHandler.handle(cmd)`: Use kg.count_nodes(), count_edges(), count_locked(), count_pending_contradictions(). Compute graph_hash via RegressionChecker.capture_metrics(). Return KnowledgeStatusResult.
- `ContradictionListHandler.handle(cmd)`: Use ContradictionManager.list_all(status=cmd.status, limit=cmd.limit). Return ContradictionListResult.
- `ContradictionResolveHandler.handle(cmd)`: Use ContradictionManager.resolve(cmd.contradiction_id, cmd.resolution, cmd.merge_value). Return ContradictionResolveResult.
- `FactLockHandler.handle(cmd)`: If cmd.action == 'lock', use FactLockManager.owner_confirm_lock(). If 'unlock', use FactLockManager.unlock_fact(). Return FactLockResult.
- `KnowledgeRegressionHandler.handle(cmd)`: Use RegressionChecker. If cmd.snapshot_path provided, load that snapshot's kg_metrics and compare with current. If not provided, just capture and return current metrics. Return KnowledgeRegressionResult.
