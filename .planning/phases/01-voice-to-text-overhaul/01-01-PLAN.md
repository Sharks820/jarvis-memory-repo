Create new file `engine/src/jarvis_engine/stt_vad.py` with a `SileroVADDetector` class:

1. **Constructor** `__init__(self, threshold: float = 0.5, sampling_rate: int = 16000)`:
   - Store threshold and sampling_rate
   - `_model = None` (lazy loaded)
   - Call `torch.set_num_threads(1)` on first model load per research pitfall 3 (prevents PyTorch thread contention with sentence-transformers)

2. **`_ensure_model(self)`**: Lazy-load Silero VAD via `from silero_vad import load_silero_vad`. Wrap in try/except ImportError, log warning and set `_model = None` if not installed.

3. **`is_speech(self, audio_chunk: np.ndarray) -> bool`**: Takes a float32 numpy chunk (512 samples at 16kHz). Returns True if speech confidence > threshold. Converts to `torch.FloatTensor`, calls `self._model(tensor, self._sampling_rate).item()`. Returns False if model not available.

4. **`get_confidence(self, audio_chunk: np.ndarray) -> float`**: Same as is_speech but returns the raw probability. Returns 0.0 if model not available.

5. **`process_chunk(self, audio_chunk: np.ndarray) -> bool`**: Handles chunks larger than 512 samples by splitting into 512-sample sub-windows and returning True if ANY sub-window has speech (max confidence across sub-windows > threshold). This handles the wakeword.py 1280-frame chunk case per research pitfall 2/open question 4.

6. **`reset(self)`**: Calls `self._model.reset_states()` if model loaded. Critical per research pitfall 5 (stateful model carries over state between utterances).

7. **`available` property**: Returns True if torch and silero_vad are importable.

Add module-level `_vad_instance: SileroVADDetector | None` singleton with lock for shared use (same pattern as `_local_stt_instance` in stt.py).
