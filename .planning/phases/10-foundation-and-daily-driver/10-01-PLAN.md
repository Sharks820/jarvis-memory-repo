  <action>
    **BiometricHelper.kt:**
    - Object (singleton) with `authenticate(activity: FragmentActivity, onSuccess: () -> Unit, onFailure: (String) -> Unit)` function
    - Uses BiometricPrompt.PromptInfo.Builder() with:
      - setTitle("Jarvis Authentication")
      - setSubtitle("Verify your identity")
      - setAllowedAuthenticators(BIOMETRIC_STRONG or BIOMETRIC_WEAK or DEVICE_CREDENTIAL)
      - setConfirmationRequired(false) for fast unlock
    - BiometricPrompt callback: onAuthenticationSucceeded calls onSuccess, onAuthenticationError/onAuthenticationFailed calls onFailure with error message
    - `canAuthenticate(context: Context): Boolean` checks BiometricManager.canAuthenticate() returns BIOMETRIC_SUCCESS
    - All operations run on main thread (BiometricPrompt requirement)

    **CryptoHelper.kt:**
    - Object with lazy-initialized EncryptedSharedPreferences instance
    - `getEncryptedPrefs(context: Context): SharedPreferences` creates EncryptedSharedPreferences with:
      - MasterKey.Builder(context).setKeyScheme(MasterKey.KeyScheme.AES256_GCM).build()
      - EncryptedSharedPreferences.PrefKeyEncryptionScheme.AES256_SIV
      - EncryptedSharedPreferences.PrefValueEncryptionScheme.AES256_GCM
    - Store/retrieve functions: `saveSigningKey(context, key)`, `getSigningKey(context): String?`, `saveAuthToken(context, token)`, `getAuthToken(context): String?`, `saveDesktopUrl(context, url)`, `getDesktopUrl(context): String?`
    - `getDatabasePassphrase(context: Context): ByteArray` -- generates a random 32-byte passphrase on first call, stores it in EncryptedSharedPreferences, returns it on subsequent calls. This is the SQLCipher key.
    - `verifyMasterPassword(context: Context, password: String): Boolean` -- checks against stored hash using PBKDF2WithHmacSHA256 (200000 iterations, matching desktop owner_guard.py). Returns false if no master password is set.
    - `setMasterPassword(context: Context, password: String)` -- generates random salt, hashes with PBKDF2, stores hash + salt in EncryptedSharedPreferences
    - `isMasterPasswordSet(context: Context): Boolean`

    **JarvisDatabase.kt:**
    - @Database annotation with entities: CommandQueueEntity, ConversationEntity, SettingsEntity. Version = 1, exportSchema = false.
    - Abstract class extending RoomDatabase with abstract dao accessor functions
    - Hilt @Module providing singleton JarvisDatabase:
      - Gets passphrase from CryptoHelper.getDatabasePassphrase(context)
      - Creates SupportFactory(passphrase) from net.zetetic.database.sqlcipher
      - Room.databaseBuilder(context, JarvisDatabase::class.java, "jarvis_encrypted.db")
        .openHelperFactory(supportFactory)
        .build()
    - Separate @Provides for each DAO

    **CommandQueueEntity.kt:**
    - @Entity(tableName = "command_queue")
    - Fields: id (Long, auto-generate PK), command (String -- the text command), status (String -- "pending", "sending", "sent", "failed"), createdAt (Long -- epoch millis), sentAt (Long? -- nullable), retryCount (Int, default 0), response (String? -- nullable, desktop response)
    - This entity backs the offline command queue (FOUND-05)

    **ConversationEntity.kt:**
    - @Entity(tableName = "conversations")
    - Fields: id (Long, auto-generate PK), role (String -- "user" or "assistant"), content (String), timestamp (Long -- epoch millis), commandId (Long? -- nullable, links to CommandQueueEntity)

    **SettingsEntity.kt:**
    - @Entity(tableName = "settings")
    - Fields: key (String, PK), value (String)
    - Simple key-value store for app settings (sync interval, notification prefs, etc.)

    **CommandQueueDao.kt:**
    - @Dao interface
    - `@Insert suspend fun insert(command: CommandQueueEntity): Long`
    - `@Query("SELECT * FROM command_queue WHERE status = 'pending' OR status = 'failed' ORDER BY createdAt ASC") suspend fun getPendingCommands(): List<CommandQueueEntity>`
    - `@Query("UPDATE command_queue SET status = :status, sentAt = :sentAt, response = :response WHERE id = :id") suspend fun updateStatus(id: Long, status: String, sentAt: Long?, response: String?)`
    - `@Query("UPDATE command_queue SET retryCount = retryCount + 1, status = 'failed' WHERE id = :id") suspend fun markRetry(id: Long)`
    - `@Query("SELECT * FROM command_queue ORDER BY createdAt DESC LIMIT :limit") suspend fun getRecent(limit: Int = 50): List<CommandQueueEntity>`

    **ConversationDao.kt:**
    - @Dao interface
    - `@Insert suspend fun insert(message: ConversationEntity): Long`
    - `@Query("SELECT * FROM conversations ORDER BY timestamp DESC LIMIT :limit OFFSET :offset") suspend fun getMessages(limit: Int = 50, offset: Int = 0): List<ConversationEntity>`
    - `@Query("SELECT * FROM conversations WHERE content LIKE '%' || :query || '%' ORDER BY timestamp DESC") suspend fun search(query: String): List<ConversationEntity>`
    - `@Query("DELETE FROM conversations WHERE timestamp < :beforeTimestamp") suspend fun deleteOlderThan(beforeTimestamp: Long)`

    **SettingsDao.kt:**
    - @Dao interface
    - `@Insert(onConflict = OnConflictStrategy.REPLACE) suspend fun set(setting: SettingsEntity)`
    - `@Query("SELECT value FROM settings WHERE key = :key") suspend fun get(key: String): String?`
    - `@Query("SELECT * FROM settings") suspend fun getAll(): List<SettingsEntity>`
    - `@Query("DELETE FROM settings WHERE key = :key") suspend fun delete(key: String)`
