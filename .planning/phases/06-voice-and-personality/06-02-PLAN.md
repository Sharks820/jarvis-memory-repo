# Plan 06-02: Whisper-Grade Speech-to-Text Pipeline

## Objective

Add local speech-to-text using faster-whisper with microphone capture via sounddevice. Create stt.py module with lazy model loading, wire VoiceListenCommand through the Command Bus, and preserve existing voice.py Edge-TTS output unchanged.

## Requirements Covered

- VOICE-01: Edge-TTS British male neural voice output preserved (no changes to voice.py)
- VOICE-04: Whisper-grade STT processes voice commands with accuracy comparable to Whisper desktop app

## Tasks

### Task 1: Create stt.py speech-to-text module

**Files to create:**
- `engine/src/jarvis_engine/stt.py` (NEW)

**What to implement:**

1. `TranscriptionResult` dataclass:
   - `text: str`
   - `language: str`
   - `confidence: float`
   - `duration_seconds: float`

2. `SpeechToText` class:
   - Constructor: `model_size: str = "small.en"`, `device: str = "cpu"`, `compute_type: str = "int8"`
   - Read `JARVIS_STT_MODEL` env var to override model_size
   - `_model` attribute (lazy, None until first use)
   - `_ensure_model()` -- lazy-load with `from faster_whisper import WhisperModel` inside try/except ImportError
   - `transcribe_audio(audio, *, language="en", vad_filter=True) -> TranscriptionResult` -- accepts numpy array or file path string
   - Handle empty transcription (return result with empty text)

3. `record_from_microphone(*, sample_rate=16000, max_duration_seconds=30.0) -> np.ndarray`:
   - Import sounddevice inside function (lazy)
   - Catch PortAudioError with clear message about Windows microphone permissions
   - Record mono float32 at 16kHz
   - Return flattened numpy array

4. `listen_and_transcribe(*, max_duration_seconds=30.0, language="en", model_size="small.en") -> TranscriptionResult`:
   - Convenience function: record_from_microphone() -> transcribe_audio()
   - Single entry point for voice command capture

**Graceful degradation:**
- If faster-whisper not installed, raise RuntimeError with install instructions
- If sounddevice not installed, raise RuntimeError with install instructions
- If no microphone available, return TranscriptionResult with error in text field

**Bus registration in app.py:**

### Task 3: CLI subcommand for voice-listen

**Files to modify:**
- `engine/src/jarvis_engine/main.py` (MODIFY -- add voice-listen subcommand)

**Subcommand: `voice-listen`**
- Arguments: `--duration` (default 30), `--language` (default en), `--model` (default small.en), `--execute` (flag)
- Dispatches VoiceListenCommand through bus
