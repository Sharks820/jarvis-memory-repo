## Tasks

### Task 1: Extend growth_tracker.py with memory-recall golden tasks

**Files to modify:**
- `engine/src/jarvis_engine/growth_tracker.py` (MODIFY)

**Add dataclasses:**
1. `MemoryRecallTask`: task_id, query, must_find_branches (list[str]), min_results (int), must_include_in_results (list[str])
2. `MemoryRecallResult`: task_id, query, results_found, branches_found (list[str]), branch_coverage (float), keyword_coverage (float), overall_score (float)

**Add functions:**
1. `evaluate_memory_recall(task, engine, embed_service) -> MemoryRecallResult`
   - Embed query via embed_service.embed(query, prefix="search_query")
   - Search via engine.search_vec(embedding, limit=10)
   - Score: has_results (0.3) + branch_coverage (0.3) + keyword_coverage (0.4)
   - Branches detected from record.get("branch", "general")
2. `run_memory_eval(tasks, engine, embed_service) -> list[MemoryRecallResult]`
   - Evaluate all tasks, return list of results
   - If engine or embed_service is None, raise RuntimeError

**Default memory-recall tasks (5 tasks):**
```python
DEFAULT_MEMORY_TASKS = [
    MemoryRecallTask("health_recall", "What medications does the owner take?", ["health"], 1, ["medication"]),
    MemoryRecallTask("gaming_recall", "What games does the owner play?", ["gaming"], 1, ["game"]),
    MemoryRecallTask("ops_recall", "What are the owner's upcoming tasks?", ["ops"], 1, ["task"]),
    MemoryRecallTask("family_recall", "Tell me about the owner's family", ["family"], 1, ["family"]),
    MemoryRecallTask("coding_recall", "What programming projects is the owner working on?", ["coding"], 1, ["code"]),
]
```

### Task 2: Add knowledge growth metrics capture

**Files to create:**

**Function:**
- `capture_knowledge_metrics(kg, engine) -> dict`
  - Count total records via engine (search_fts with empty query or direct SQL)
  - Count total facts, edges, locked facts from KG
  - Count branch distribution from records table
  - Count temporal distribution from kg_nodes (try/except for missing columns)
  - Return dict with: total_records, total_facts, total_edges, locked_facts, branches_populated, branch_distribution, temporal_distribution, captured_at

### Task 3: Add CLI subcommands for learning operations

**Files to modify:**
- `engine/src/jarvis_engine/main.py` (MODIFY)

**New subcommands:**
1. `learn` -- Manually trigger learning from text input
   - Arguments: `--user-message TEXT`, `--assistant-response TEXT`
   - Dispatches LearnInteractionCommand through bus
   - Prints records_created count

2. `cross-branch-query` -- Query across branches
   - Arguments: `query` (positional), `--k` (default 10)
   - Dispatches CrossBranchQueryCommand through bus
   - Prints direct results and cross-branch connections

3. `flag-expired` -- Flag expired facts
   - No arguments
   - Dispatches FlagExpiredFactsCommand through bus
   - Prints expired_count

4. `memory-eval` -- Run memory-recall golden task evaluation
   - Arguments: none
   - Runs run_memory_eval() with DEFAULT_MEMORY_TASKS
   - Prints per-task scores and overall average
