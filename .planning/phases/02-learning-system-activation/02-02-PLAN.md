After computing `sim` for each route but BEFORE comparing to `best_sim`, apply the quality penalty:
   ```python
   # Apply route quality penalty (LEARN-02)
   if self._feedback_tracker is not None:
       try:
           quality = self._feedback_tracker.get_route_quality(route_name)
           if quality["total"] >= 5:  # Minimum sample threshold
               # Scale similarity by 0.5-1.0 based on satisfaction rate
               # 100% satisfaction = no penalty, 0% = halved similarity
               sim *= (0.5 + 0.5 * quality["satisfaction_rate"])
       except Exception:
           pass  # Graceful degradation -- no penalty on error
   ```
   CRITICAL: The minimum threshold of 5 prevents the penalty from activating on sparse data. The floor of 0.5x prevents any route from being fully blocked.
