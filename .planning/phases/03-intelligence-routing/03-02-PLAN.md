    1. Create `engine/src/jarvis_engine/gateway/classifier.py`:
       - Import `numpy as np` (already a project dependency).
       - Define `IntentClassifier` class:
         - Class constant `ROUTES: dict[str, list[str]]` with three keys:
           - `"complex"`: 5 exemplars for complex reasoning/coding tasks. Examples: "write a Python script that implements a binary search tree with balancing", "analyze this codebase and suggest architectural improvements", "help me debug this race condition in my threading code", "explain the tradeoffs between CQRS and event sourcing", "review this security policy and identify vulnerabilities".
           - `"routine"`: 5 exemplars for summarization/translation/formatting. Examples: "summarize this article for me", "rewrite this paragraph to be more concise", "what are the key points from this meeting transcript", "translate this text to French", "format this data as a markdown table".
           - `"simple_private"`: 5 exemplars for private/calendar/personal data. Examples: "what's on my calendar today", "what medications do I take", "remind me about my doctor appointment", "what did I have for dinner yesterday", "show me my recent bills".
         - Class constant `MODEL_MAP: dict[str, str]`:
           - `"complex"` -> `"claude-opus-4-5-20250929"`
           - `"routine"` -> `"claude-sonnet-4-5-20250929"`
           - `"simple_private"` -> configurable via `JARVIS_LOCAL_MODEL` env var, default `"qwen3:14b"`
         - Class constant `PRIVACY_KEYWORDS: set[str]` = `{"calendar", "medication", "medications", "medicine", "pill", "prescription", "bill", "bills", "payment", "password", "personal", "private", "salary", "bank", "account", "doctor", "appointment", "family", "wife", "husband", "son", "daughter", "address", "phone number", "social security"}`.
         - Class constant `CONFIDENCE_THRESHOLD: float = 0.35` -- if the best route's cosine similarity is below this, default to local.
         - `__init__(self, embed_service)`: Store the EmbeddingService. Call `_precompute_routes()` to build centroid embeddings.
         - `_precompute_routes(self) -> dict[str, np.ndarray]`: For each route, embed all exemplars using `embed_service.embed(text, prefix="search_query")` and compute the mean (centroid) of all exemplar embeddings. Return dict mapping route name -> centroid ndarray.
         - `_check_privacy(self, query: str) -> bool`: Return True if any privacy keyword appears in the lowercased query. Use word boundary matching (check if keyword is a substring of the lowered query -- simple `keyword in query_lower` is sufficient since the keywords are specific enough).
         - `classify(self, query: str) -> tuple[str, str, float]`: Return (route_name, model_name, confidence).
           - FIRST: Check `_check_privacy(query)`. If True, return `("simple_private", MODEL_MAP["simple_private"], 1.0)`.
           - Embed the query with `embed_service.embed_query(query)`.
           - Compute cosine similarity against each route centroid.
           - Find the best route (highest similarity).
           - If best similarity < `CONFIDENCE_THRESHOLD`, default to `("simple_private", MODEL_MAP["simple_private"], best_similarity)` -- when uncertain, keep it local.
           - Otherwise return the best route, its model, and the similarity score.
         - Static method `_cosine_sim(a, b) -> float`: `np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))`.
       - Also export IntentClassifier from `gateway/__init__.py`.
