    - save_error_fix(error_message, fix_description, code_before, code_after) stores a KG fact with node_type="error_fix", node_id="errfix:{hash}", label containing error summary + fix description, confidence=0.8, source_record="compile_fix_loop"
    - query_patterns(task_description, limit=5) calls KG semantic search for node_type in ("code_pattern", "error_fix") and returns a list of strings suitable for prompt injection

    - Code snippets in labels are truncated to 500 chars to avoid bloating the KG
