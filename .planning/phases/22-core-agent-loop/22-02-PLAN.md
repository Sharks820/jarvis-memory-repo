  <behavior>
    StepExecutor:
    - execute_step(step, task) looks up tool via registry.get(step.tool_name), calls spec.execute(**step.params)
    - execute_step checkpoints task to state_store BEFORE calling tool
    - execute_step emits step_start event before and step_done event after tool call
    - execute_step returns StepResult(success=True, output=...) on success
    - execute_step returns StepResult(success=False, error=...) on tool exception
    - execute_step checks ApprovalGate before executing; if REQUIRES_APPROVAL, waits for approval
    - execute_step sets task.status="blocked" and task.approval_needed=True when waiting

    ReflectionLoop:
    - evaluate(step_result) returns True if step succeeded
    - run_loop(task, steps) executes steps in order via StepExecutor, calling evaluate after each
    - On failure: calls planner.replan() with error context, replaces remaining steps
    - Tracks consecutive same-error count via error hash (hash of last_error string)
    - After 3 consecutive same-error failures: sets task.status="failed", returns with escalation message
    - Enforces token_budget: if task.tokens_used >= task.token_budget, stops and escalates
    - On success: advances step_index, continues to next step
    - On all steps complete: sets task.status="done"
  </behavior>
  <action>
    **StepResult dataclass** (executor.py):
    ```python
    @dataclass
    class StepResult:
        success: bool
        output: str = ""
        error: str = ""
        tokens_used: int = 0
    ```

    **StepExecutor** (executor.py):
    - __init__(self, registry: ToolRegistry, gate: ApprovalGate, store: AgentStateStore, bus: ProgressEventBus)
    - execute_step(self, step: AgentStep, task: AgentTask) -> StepResult:
      1. Look up tool: spec = registry.get(step.tool_name). If None, return StepResult(success=False, error="Unknown tool").
      2. Check approval: decision = gate.check(spec, step.params). If REQUIRES_APPROVAL, set task.status="blocked", task.approval_needed=True, store.checkpoint(task), emit approval_needed event, await gate.wait_for_approval(). If rejected, return failure.
      3. Checkpoint: task.status="running", store.checkpoint(task).
      4. Emit step_start event.
      5. Call tool: result = await spec.execute(**step.params). Handle sync tools by checking if result is coroutine.
      6. Emit step_done event.
      7. Return StepResult(success=True, output=str(result)).
      8. On exception: return StepResult(success=False, error=str(e)).

    **ReflectionLoop** (reflection.py):
    - __init__(self, executor: StepExecutor, planner: TaskPlanner, store: AgentStateStore, bus: ProgressEventBus)
    - run_loop(self, task: AgentTask, steps: list[AgentStep]) -> AgentTask:
      1. Start from task.step_index.
      2. For each step:
         a. Check token budget: if task.tokens_used >= task.token_budget, set failed, emit budget_exceeded, return.
         b. Execute step via executor.execute_step().
         c. If success: advance task.step_index, checkpoint, continue.
         d. If failure: hash the error, compare to previous error hash.
            - If same error 3 times: set task.status="failed", task.last_error=error, checkpoint, emit escalation event, return.
            - Otherwise: increment task.error_count, call planner.replan() with error context, replace remaining steps, update task.plan_json, checkpoint, retry from current index.
      3. All steps done: task.status="done", checkpoint, emit task_done event, return.
    - _error_hash(self, error: str) -> str: hashlib.md5 of the error string (for dedup, not security).

    **Key design decisions:**
    - The loop is synchronous at the top level (runs in ThreadPoolExecutor from handler). Tool execute calls that are async are run via asyncio.run() or a dedicated event loop. DECISION: Use a running asyncio loop. The handler creates an event loop for the agent thread.
    - Max replan attempts per step: 3 (matching the 3-consecutive-same-error rule).
    - Token tracking: planner.plan/replan returns tokens_used, added to task.tokens_used.
