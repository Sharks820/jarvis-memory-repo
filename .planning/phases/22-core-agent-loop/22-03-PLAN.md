  <behavior>
    AgentRunHandler:
    - handle(cmd) creates AgentTask with goal, task_id (uuid if empty), token_budget
    - handle(cmd) checkpoints task to store with status="pending"
    - handle(cmd) launches _run_agent_loop in ThreadPoolExecutor (non-blocking)
    - handle(cmd) returns AgentRunResult with task_id and status="pending"
    - _run_agent_loop plans steps via TaskPlanner, then runs ReflectionLoop.run_loop

    AgentStatusHandler:
    - handle(cmd) loads task from AgentStateStore
    - Returns status, step_index, tokens_used, last_error from stored task
    - Returns return_code=1 if task not found

    AgentApproveHandler:
    - handle(cmd) calls gate.approve(task_id) if approved=True, gate.reject(task_id) if False
    - Returns action_taken description

    SSE route (/agent/stream):
    - Subscribes to ProgressEventBus
    - Streams events as text/event-stream SSE format
    - Sends keep-alive pings every 30s
    - Unsubscribes on disconnect
  </behavior>
  <action>
    **agent_handlers.py** — Replace stub implementations:

    AgentRunHandler.__init__ now accepts: root, gateway, registry, store, gate, bus (all the subsystem objects).
    AgentRunHandler.handle(cmd):
    1. Generate task_id = cmd.task_id or secrets.token_hex(8).
    2. Create AgentTask(task_id=task_id, goal=cmd.goal, token_budget=cmd.token_budget, status="pending").
    3. store.checkpoint(task).
    4. Submit _run_agent_loop(task) to a ThreadPoolExecutor(max_workers=1) stored as class attribute.
    5. Return AgentRunResult(return_code=0, message="Task submitted", task_id=task_id, status="pending").

    _run_agent_loop(task):
    1. Create event loop for this thread: loop = asyncio.new_event_loop().
    2. planner = TaskPlanner(gateway, registry).
    3. executor_obj = StepExecutor(registry, gate, store, bus).
    4. reflection = ReflectionLoop(executor_obj, planner, store, bus).
    5. steps, tokens = planner.plan(task.goal).
    6. task.plan_json = json.dumps([asdict(s) for s in steps]).
    7. task.tokens_used += tokens. task.status = "running". store.checkpoint(task).
    8. loop.run_until_complete(reflection.run_loop(task, steps)).

    AgentStatusHandler.__init__ accepts: root, store.
    AgentStatusHandler.handle: Load task from store, return fields.

    AgentApproveHandler.__init__ accepts: root, gate.
    AgentApproveHandler.handle: Call gate.approve or gate.reject.

    **app.py** — Update _register_agent_handlers to pass subsystem objects:
    - Create ProgressEventBus singleton.
    - Create AgentStateStore using existing db connection (from engine).
    - Create ApprovalGate with bus.
    - Pass gateway, registry, store, gate, bus to handler constructors.
    - Register built-in tools (FileTool, ShellTool, WebTool, UnityTool) on the registry.
    - Store registry and bus on a module-level or pass through.
