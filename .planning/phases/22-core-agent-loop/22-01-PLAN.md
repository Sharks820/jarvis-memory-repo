<task type="auto" tdd="true">
  <name>Task 2: ApprovalGate and ProgressEventBus</name>
  <files>
    engine/src/jarvis_engine/agent/approval_gate.py

    engine/tests/test_approval_gate.py

  <behavior>
    ApprovalGate:
    - check(tool_spec, params) returns ApprovalDecision.AUTO for safe tools (requires_approval=False)
    - check(tool_spec, params) returns ApprovalDecision.REQUIRES_APPROVAL for destructive tools
    - check(tool_spec, params) returns ApprovalDecision.REQUIRES_APPROVAL for costly tools (estimate_cost > 0)
    - approve(task_id) resolves the pending approval future
    - reject(task_id) resolves the pending approval future with rejected=True
    - wait_for_approval(task_id) blocks until approve/reject is called

    ProgressEventBus:
    - emit(event_dict) puts event on bounded queue (max 256 items)
    - subscribe() returns an asyncio.Queue that receives all future events
    - unsubscribe(queue) removes a subscriber
    - emit with full queue drops oldest event (no block)
    - Multiple subscribers each get a copy of every event
  </behavior>
  <action>
    **ApprovalGate** (approval_gate.py):
    - Enum ApprovalDecision: AUTO, REQUIRES_APPROVAL
    - Class ApprovalGate:
      - __init__(self, progress_bus: ProgressEventBus): stores bus reference for emitting approval_needed events.
      - check(self, spec: ToolSpec, params: dict) -> ApprovalDecision: If spec.requires_approval or spec.is_destructive, return REQUIRES_APPROVAL. If spec.estimate_cost(**params) > 0, return REQUIRES_APPROVAL. Otherwise AUTO.
      - async wait_for_approval(self, task_id: str, step_desc: str) -> bool: Create asyncio.Event, emit approval_needed event via bus, await the event. Return True if approved, False if rejected.
      - approve(self, task_id: str) -> None: Set the event + store approved=True.
      - reject(self, task_id: str) -> None: Set the event + store approved=False.
      - Internal dict _pending: dict[str, tuple[asyncio.Event, bool]] for tracking.

    **ProgressEventBus** (progress_bus.py):
    - Class ProgressEventBus:
      - __init__(self, max_size: int = 256): Create internal subscriber list.
      - async emit(self, event: dict) -> None: For each subscriber queue, put_nowait. If queue full, get_nowait to drop oldest, then put_nowait.
      - subscribe(self) -> asyncio.Queue: Create new Queue(maxsize=256), add to subscribers, return it.
      - unsubscribe(self, queue: asyncio.Queue) -> None: Remove from subscribers list.
      - Module-level singleton: _bus: ProgressEventBus | None = None; get_progress_bus() -> ProgressEventBus.
  </action>
