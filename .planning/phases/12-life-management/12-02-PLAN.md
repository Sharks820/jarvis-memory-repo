      - Chase: `(?i)(?:purchase|charge|transaction).*?\$([\\d,]+\\.\\d{2}).*?at\\s+(.+?)(?:\\.|$)`

Also checks if package contains "bank", "finance", "credit".

    **AnomalyDetector** (`feature/finance/AnomalyDetector.kt`): Hilt `@Singleton` class injecting `TransactionDao`. Provides:
    - `suspend fun check(transaction: TransactionEntity): AnomalyResult` with `data class AnomalyResult(val isAnomaly: Boolean, val reason: String)`.
    - Checks three anomaly types:
      1. **Unusual amount**: Gets average spend for this category over last 90 days via `transactionDao.getAverageAmountForCategory()`. If transaction amount > 3x the average (and average > $0), flag as anomaly: "Unusual charge: ${amount} is ${multiplier}x your average ${category} spend".

    - If anomaly detected, posts an IMPORTANT priority notification via `NotificationCompat.Builder` on `jarvis_important` channel with title "Financial Alert" and body = anomaly reason.
